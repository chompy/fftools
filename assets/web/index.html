FFTOOLS

Lua scripting in Final Fantasy XIV!
About

FFTools is an Advanced Combat Tracker (ACT) plugin that enables the use of Lua scripts to create complex text-to-speech triggers, sharable web views, and more!

It offers much of the same functionality of tools like Cactbot and Triggernometry. However, FFTools allows users to install only the scripts they need and is 100% open source. So you know exactly what you're installing on your machine.
